🎬 Filmovita - LV1 Zadatak

✅ Projekt je uspješno kreiran!
Sve potrebne datoteke su generirane i spremne za korištenje.

📁 Kreirani fajlovi

  • index.html - Glavna stranica s tablicom filmova
  • grafikon.html - Stranica s CSS-only grafikonima
  • slike.html - Galerija s lightbox efektom
  • style.css - Zajednički stilovi
  • style-index.css - Stilovi za index.html
  • style-grafikon.css - Stilovi za grafikon.html
  • style-slike.css - Stilovi za slike.html
  • README.md - Dokumentacija projekta
  • UPUTE.html - Ovaj fajl s uputama

🚀 Kako pokrenuti projekt

Korak 1: Preuzmite sve fajlove u istu mapu na vašem računalu
Korak 2: Dvostrukim klikom otvorite index.html u web pregledniku
Korak 3: Koristite navigaciju za prelazak između stranica:
  • Početna - index.html
  • Grafikoni - grafikon.html
  • Galerija - slike.html

🌐 Objava na internetu

Za objavu projekta na internetu, koristite jedan od sljedećih servisa:

GitHub Pages (preporučeno):
  1. Kreirajte GitHub repozitorij
  2. Uploadajte sve fajlove
  3. Idite na Settings > Pages
  4. Odaberite branch (main) i Save
  5. Vaša stranica će biti dostupna na: https://vase-ime.github.io/naziv-repozitorija/
📖 GitHub Pages dokumentacija
Netlify Drop:
  1. Posjetite netlify.com/drop
  2. Prevucite mapu s fajlovima na stranicu
  3. Dobijete instant live link
Vercel:
  1. Instalirajte Vercel CLI: npm i -g vercel
  2. U mapi projekta pokrenite: vercel
  3. Pratite upute za deployment

  • ✅ Semantički HTML s ARIA atributima
  • ✅ SEO meta tagovi
  • ✅ Dropdown navigacija bez JavaScripta
  • ✅ Responzivna tablica filmova

slike.html

  • ✅ Grid galerija slika
  • ✅ Lightbox efekt bez JavaScripta (:target pseudo-klasa)

  • ✅ Zoom efekt pri hoveru
  • ✅ Responzivni prikaz

🎨 Tehnike korištene u projektu

Sve bez JavaScripta! Projekt je izgrađen isključivo s HTML-om i CSS-om, koristeći napredne CSS tehnike:
  • :checked pseudo-klasa za dropdown menu
  • :target pseudo-klasa za lightbox
  • conic-gradient za pie chart
  • CSS Grid i Flexbox za layout
  • CSS animacije i transformacije
  • Media queries za responzivnost
  • CSS varijable za lako održavanje

🧪 Testiranje

Testirajte projekt na:

  • Desktop: Chrome, Firefox, Safari, Edge
  • Tablet: iPad, Android tableti (koristi DevTools)
  • Mobile: iPhone, Android telefoni (koristi DevTools)

Alati za testiranje pristupačnosti:

  • ChromeVox: Screen reader ekstenzija za Chrome
  • WAVE: Web accessibility evaluation tool
  • Lighthouse: Ugrađen u Chrome DevTools
